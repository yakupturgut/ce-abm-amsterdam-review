# Results folder

Generated simulation outputs, tables and figures are written here.

The repository intentionally tracks only placeholder files by default. Run outputs
can be large and should be regenerated or added deliberately.

Subfolders:

- `runs/`: per-run monthly metrics and compact spatial arrays;
- `experiments/`: replicated experiment output groups;
- `tables/`: manifests, summaries and diagnostics;
- `figures/`: generated manuscript and diagnostic figures.
