# Data folder

- `raw/` contains raw spatial and socioeconomic input files used for the Amsterdam case implementation.
- `external/` contains supplementary external data, including repair-cafe locations.
- `processed/` contains files generated by `src/preprocess_amsterdam.py` and used directly by the ABM.

The preprocessed files are included so reviewers can run the model without first
rerunning the geospatial preprocessing workflow.
