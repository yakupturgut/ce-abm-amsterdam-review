# Reproducibility checklist

Before sharing the repository link, check the following.

- [ ] Repository has no author names, affiliations, ORCID IDs or personal emails.
- [ ] Repository has no local paths such as `C:/Users/...`, OneDrive paths or Desktop paths.
- [ ] `README.md` explains model purpose, installation, quick run and full run order.
- [ ] `requirements.txt` is present.
- [ ] `data/processed/` contains the preprocessed model inputs needed for a quick run.
- [ ] `results/` contains only `.gitkeep` and generated outputs are not committed unless intentionally added.
- [ ] Quick demo run works with `CE_ABM_QUICK_DEMO=1`.
- [ ] Main figure plotting works after a demo or full replicated run.
- [ ] Sensitivity outputs are either regenerated or added intentionally.
- [ ] Code-availability statement in the manuscript uses the final repository URL.
